
# Приложения, обеспечивающего настройку музыкальных инструментов 
Проект создан студентом группы 8В32 Романчук Софьей
## Для создания проекта использовались:
* Стреда разработки Qt Creator
* Qt 6.8.2 (shared) MinGW-w64 MINGW64 (MSYS2)
* Библиотека aubio (https://aubio.org/)
* Стандарт C++: C++17
* Компилятор MinGW 11.2 (64-bit) / GCC 11+

## Требования к составу и параметрам технических средств
* процессор Intel Core i3 или AMD Ryzen 3 (или совместимый) с тактовой частотой 2.0 ГГц или выше,
* объем свободной оперативной памяти 4 ГБ ОЗУ,
* объем необходимой памяти на жестком диске 30 Мб,
* стандартный VGA монитор,
* стандартная клавиатура,
* манипулятор «мышь»,
* микрофон с 1 каналом.

## Сборка проекта на Windows
* Установка Qt 6: pacman -S mingw-w64-x86_64-qt6
* Установка Qt Creator: pacman -S mingw-w64-x86_64-qt-creator
* Установка aubio: pacman -S mingw-w64-x86_64-aubio
* Клонировать репозиторий: git clone https://github.com/smtenjoyer/Tuner/
* Удалить папку build если она есть.
* Открыть проекты tuner в qtcreator. Для этого необходимо открыть tuner.pro.
* Комплект сборки: Desktop Qt 6.8.2 MinGW 64-bit
* Собрать проект. Для этого на левой панели нужно выбрать проекты и настроить сборку.
* Конфигурация сборки:выпуск
